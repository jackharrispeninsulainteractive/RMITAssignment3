Timeframe Description

To handle our timeframe I have created a interactive scrollable timeline that clearly lays out the tasks that are archive for each team members to tackle each week, some tasks, if large can be assigned to start and finish over multiple weeks. During this time line you will constantly see tasks that start one week and finish the next week, this allows us to manage larger asks as well as weekly tasks on the same time line.

In addition to this list each week will also have a list of tasks that must be completed that week, these can be from the week prior or 2 weeks prior depending on the duration of the task.
